# Q3-oriented preliminary C-band LaTeX paper

This branch was created to hold the Q3-oriented paper draft for the ISRS-SCL project.

Recommended manuscript title:

**A Preliminary GNPy-Assisted External Validation Workflow for ISRS-Aware GSNR Modeling in C-Band Coherent Optical Links**

Scope:

- This is a preliminary C-band validation paper, not a full S+C+L paper.
- Evidence used: 9 clean GNPy C-band rows at 1535, 1550, and 1560 nm across 1, 4, and 8 spans.
- External-only validation coverage: 9/9 matched rows.
- Raw GSNR RMSE: 1.762 dB.
- Raw bias: -1.612 dB.
- Wavelength-dependent diagnostic leave-one-out RMSE: 0.228 dB.
